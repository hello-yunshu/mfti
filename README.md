# MFTI - Male to Female Type Indicator

一个「男娘」人格测试，纯属娱乐。37道题目，10个维度评分，测出你是哪种生物。

👉 **立即体验：[https://mfti.hey.run/](https://mfti.hey.run/)**

## 项目简介

MFTI 是一个基于 Myers-Briggs Type Indicator (MBTI) 概念的趣味人格测试，但专为跨性别女性 (MTF) 和「男娘」群体设计。

- **科学性**：约等于星座，但比星座有趣
- **娱乐性质**：整活性质，不要认真
- **16种人格**：每种人格都有独特的心理学画像和描述

## 功能特性

- 🎯 **37道测试题**：精心设计的题目，覆盖多个维度
- 📊 **10个维度评分**：粉色梦境指数、社交活跃度、颜值焦虑度、公开度等
- 🌸 **16种人格类型**：每种都有独特的emoji、配色和详细描述
- 🔮 **相似人格推荐**：显示与你最相似的其他3种人格
- 💾 **本地存储**：自动保存测试结果，下次打开直接查看

## 10个维度

| 维度 | 代码 | 说明 |
|------|------|------|
| 粉色梦境指数 | PR | 你活在粉色滤镜里还是现实里 |
| 社交活跃度 | SH | 你是社牛还是宅神 |
| 颜值焦虑度 | AI | 你对自身外貌有多焦虑 |
| 公开度 | EW | 你出来了没有 |
| 自律指数 | CF | 你是计划派还是随缘党 |
| 社群归属感 | DK | 你需要群体还是一个人就够了 |
| 理性分析度 | TN | 你分析身份还是跟着感觉走 |
| 物质投入度 | MB | 你在这件事上花了多少钱 |
| 表达一致性 | GL | 你每天都在状态还是看心情 |
| 未来规划度 | XZ | 你有长期规划还是走一步看一步 |

## 4字母编码规则

| 位置 | 维度 | 左（>50） | 右（<50） |
|------|------|-----------|-----------|
| 1 | PR | P（梦幻派） | R（现实派） |
| 2 | SH | S（社交蝴蝶） | H（宅家精灵） |
| 3 | AI | A（颜值焦虑） | W（心态平和） |
| 4 | EW | E（公开出柜） | I（未出柜） |
